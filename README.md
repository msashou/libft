## Description
libft課題は、C標準ライブラリ（libc）のいくつかの有用な関数を、ft_形式で再実装するものである。

## Instructions
コンパイルは cc -Wall -Werror -Wextra を使用して実行する。  
Makefileを使用する場合、make all で全ファイルのコンパイル、make clean で全.oファイルの削除、make fclean で.aファイルの削除が可能である。

## Functions

### Part 1 - Libc functions
- `ft_isalpha` : 文字が英字であるか判定する。
- `ft_isdigit` : 文字が数字（0〜9）であるか判定する。
- `ft_isalnum` : 文字が英字または数字であるか判定する。
- `ft_isascii` : 文字が標準ASCII文字セットに適合するか判定する。
- `ft_isprint` : 文字がスペースを含む表示可能文字であるか判定する.
- `ft_strlen` : 終端ヌルバイトを除く文字列の長さを計算する。
- `ft_memset` : メモリ領域の先頭Nバイトを特定のバイト値で埋める。
- `ft_bzero` : メモリ領域のNバイトにゼロを書き込んでデータを消去する。
- `ft_memcpy` : 重複を考慮せず、コピー元メモリ領域からコピー先メモリ領域へNバイトコピーする。
- `ft_memmove` : 重複領域を安全に処理しながら、コピー元メモリ領域からコピー先メモリ領域へNバイトコピーする。
- `ft_strlcpy` : サイズ制限を設けて文字列をコピー先バッファにコピーし、ヌル終端を保証する。
- `ft_strlcat` : サイズ制限を設けてコピー先バッファの末尾に文字列を追加し、ヌル終端を保証する。
- `ft_toupper` : 小文字を対応する大文字に変換する。
- `ft_tolower` : 大文字を対応する小文字に変換する。
- `ft_strchr` : 文字列内で指定された文字が最初に現れる位置を特定する。
- `ft_strrchr` : 文字列内で指定された文字が最後に現れる位置を特定する。
- `ft_strncmp` : 指定されたバイト数まで2つの文字列を比較する。
- `ft_memchr` : 指定されたメモリブロック内で特定のバイトが最初に現れる位置を特定する。
- `ft_memcmp` : 2つのメモリブロックを指定されたサイズまでバイト単位で比較する。
- `ft_strnstr` : 探索する文字数を制限して、文字列内から指定された部分文字列が最初に現れる位置を特定する。
- `ft_atoi` : 整数を表す文字列を実際の整数値に変換する。
- `ft_calloc` : 配列用のメモリを割り当て、全バイトをゼロに初期化してそのポインタを返す。
- `ft_strdup` : 指定された文字列の複製に必要なメモリを割り当て、コピーを実行する。

### Part 2 - Additional functions
- `ft_substr` : 開始インデックスと最大長に基づき、指定された文字列から特定の部分文字列を割り当てて返す。
- `ft_strjoin` : 2つの入力文字列を連結した新しい文字列を割り当てて返す。
- `ft_strtrim` : 文字列の先頭および末尾から指定された文字を取り除いた複製を割り当てて返す。
- `ft_split` : 特定の区切り文字を用いて文字列を分割し、割り当てられた文字列の配列を返す。
- `ft_itoa` : 引数として渡された整数値を表す文字列を割り当てて返す。
- `ft_strmapi` : 文字列の各文字に指定された関数を適用し、メモリ割り当てを通じて新しい文字列を作成する。
- `ft_striteri` : 文字列の各文字にインデックスを渡しながら指定された関数を適用し、文字列を直接修飾する。
- `ft_putchar_fd` : 指定されたファイルディスクリプタに1文字出力する。
- `ft_putstr_fd` : 指定されたファイルディスクリプタに文字列出力する。
- `ft_putendl_fd` : 指定されたファイルディスクリプタに文字列と改行文字を出力する。
- `ft_putnbr_fd` : 整数を文字の列として指定されたファイルディスクリプタに出力する。

### Part 3 - linked list
- `ft_lstnew` : 指定されたコンテンツで初期化された新しいノードを割り当てて返す。
- `ft_lstadd_front` : 連結リストの先頭に新しいノードを追加する。
- `ft_lstsize` : 連結リストに含まれる全ノード数を数えて返す。
- `ft_lstlast` : 連結リストの最後のノードを特定して返す。
- `ft_lstadd_back` : 連結リストの末尾に新しいノードを追加する。
- `ft_lstdelone` : 指定された削除関数を用いて、単一ノードのコンテンツおよびノード自体のメモリを解放する。
- `ft_lstclear` : 指定されたノードおよびその後に続くすべての後続ノードを連結リストから削除し解放する。
- `ft_lstiter` : 連結リストを反復処理し、各ノードのコンテンツに指定された関数を適用する。
- `ft_lstmap` : 連結リストを反復処理し、各ノードのコンテンツに関数を適用して、その結果から新しいリストを作成する。
